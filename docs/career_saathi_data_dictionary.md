# Purpose

This document defines every field used in Career Saathi.

Every field must be defined here before it is used in:

- Excel Database
- Dart Models
- Firebase
- APIs
- AI Engine

Once a field is created, it should not be renamed unless absolutely necessary.
---

# Field 001

## streamId

**Purpose**

Unique identifier of the Education Stream.

**Type**

String

**Format**

STREAM_<NAME>

**Example**

STREAM_MEDICAL

**Required**

Yes

**Used In**

- Excel
- Dart Models
- Firebase
- APIs
- AI Engine
